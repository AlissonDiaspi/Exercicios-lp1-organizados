Esta coleção de programas demonstra diversas operações com arrays e matrizes em C, incluindo manipulação de matrizes bidimensionais e tridimensionais, multiplicação de matrizes e análise de desempenho.
1. ArrayBi.c
Funcionalidades:

Manipula uma matriz 2x2 usando um array unidimensional

Calcula a somatória dos elementos

Mede o tempo de execução

2. ArrayRand.c
Funcionalidades:

Gera arrays de diferentes tamanhos com valores aleatórios

Compara tempos de execução para arrays pequenos (30), médios (250) e grandes (500)

Demonstra a relação entre tamanho do array e tempo de processamento

3. ArrayTri.c
Funcionalidades:

Manipula uma matriz tridimensional (5x3x4) usando array unidimensional

Calcula a somatória dos elementos

Mede o tempo de execução

4. MultiplicaçãoArrayBi.c
Funcionalidades:

Multiplica duas matrizes 2x2 representadas como arrays unidimensionais

Mostra o resultado da multiplicação

Mede o tempo de execução



5. MultiplicaçãoArrayTri.c
Funcionalidades:

Multiplica duas matrizes tridimensionais (2x2x2) como arrays unidimensionais

Mostra o resultado da multiplicação

Mede o tempo de execução

6. MultiplicaçãoBi.c
Funcionalidades:

Multiplica matrizes bidimensionais de tamanho customizável

Verifica compatibilidade das dimensões (colunasA = linhasB)

Permite ao usuário definir as dimensões e valores das matrizes

7. MultiplicaçãoTri.c
Funcionalidades:

Multiplica matrizes tridimensionais 2x2x2

Permite ao usuário inserir valores para ambas matrizes

Exibe o resultado da multiplicação elemento por elemento
